# 沒膽的阿妹仔

文章日期：2010-07-09 21:14

<span style="color:blue">沒膽事蹟 1 － 郵局當醫院</span>
今天下午帶妹妹去郵局辦事情，想說進去吹冷氣，妹應該會覺得很舒服，沒想到，妹的表情凝重，流露出害怕的表情，我不解～ 隨著櫃檯叫號一次又一次，我發現，原來，媚妹把郵局的叫號當成是衛生所打預防針的叫號，所以，他非常害怕。當我坐在櫃檯前面辦理手續時，他也很害怕，我就跟妹說，這是領錢寄信的地方，不是打針的地方。為了鬆懈媚妹的緊繃神經，就帶妹去旁邊看郵局展售的商品，有熊貓、魚、狗狗、郵差玩偶，漸漸，他就不怕了。
<span style="color:purple">沒膽事蹟2 － 婦產科當預防注射室</span>
傍晚，我要去產檢，老公開車去驗車，所以，妹跟著我去醫院，還沒進診間，才要去拿驗尿的條子，妹就拖著我說不要進去，我帶著她去完廁所，要進去量體重和血壓，妹也堅持不進去，就兩眼害怕的眼神在門口看著我，直到我做完常規的檢查程序。之後，要進去照超音波，妹勉強乖乖坐在候診台上，but，當戴著口罩的護士又出現的時候，妹馬上大哭，他以為護士要幫他打針了，哭到不行，只好讓她躺在我的旁邊，就像平常睡覺那樣，醫生護士都覺得大眼妹哭的眼睛淚汪汪的好可愛唷！
<span style="color:blue">沒膽事蹟3 － 影子當魔鬼</span>
產檢完，才七點半，想說趁著晚風涼涼帶媚妹去公園盪鞦韆，走出大樓後，一如往常要妹"跑步"來追媽咪，但妹才剛邁出一步，馬上定住對著地上的黑影看，完全不敢動，觀察後，原來她怕影子，我跟他解釋說所有的東西有燈光照射都會產生影子，狗狗、車車、媽咪，都有影子，有影子很正常，沒影子的鬼才可怕，but，妹還是很怕，一路上，手抓的緊緊的，一直看著燈光和影子，才在走到公園的半途，妹已經嚇到一直擋在我前面要媽咪抱抱，沒辦法，我只好抱她去公園囉～～ 去到公園，盪鞦韆他還是要求要盪高高，但是，若是跟他提到地上的影子，他馬上又露出驚恐的表情，爸爸抱也不要，一直要媽媽抱，好可憐喔～ (老公還在旁邊說:這樣好有女人味唷～)，不過，爸爸堅持，爸爸抱或是自己走選一個，妹就勉強讓爸爸抱囉～
回到家，看到家中也有影子，他直嚷嚷說怕怕，不過，看起來表情沒這麼害怕了，畢竟是自己最熟悉的家。
今天一連三樁讓媚妹害怕的事情，不知明天過後他是否還記得，最好是不要再害怕影子，不然，以後都不能趁著晚風乘涼了。

<span style="color:purple">PS. 聽婆婆說，老公從小就很膽小，又很怕打針，看來，媚妹是遺傳到爸比囉~~</span>
