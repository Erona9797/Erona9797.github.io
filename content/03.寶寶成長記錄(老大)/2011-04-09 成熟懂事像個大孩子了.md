# 成熟懂事像個大孩子了

文章日期：2011-04-09 22:27

老大才兩歲半，但近日的表現讓我和婆都覺得她好像很大了。
<span style = 'color:blue'>可以道德勸說</span>：
妹妹很喜歡電子小狗(沒電不會動的)，老大看妹正在玩，跑過去搶走小狗，說:「不要給妹妹玩」。
我問：「你的玩具都是誰買給你的?」
答：「媽咪」
我說:「你借妹妹一個玩具，媽咪就送你一個玩具，你若不借妹妹，媽咪就不給你玩具」
我再問：「你要借小狗給妹妹玩嗎?」
答:「要」，並把小狗拿給妹妹。
<span style = 'color:blue'>可以威脅、利誘</span>：
1.乖乖吃飯，喝魚湯，吃完才可以玩玩具，不然媽咪要把玩具沒收，不送給你玩了唷！
2.要乖乖穿鞋鞋，腳腳乾淨才能到媽咪床上玩
3.天冷希望他穿拖鞋，就說「不穿拖鞋就把全部的鞋鞋收起來，這樣就沒有布鞋，就不能出去跑步、不能出去玩了！」
<span style = 'color:red'>自律</span>:
1.剛洗完澡，妹開心的跑著說：「我穿的衣服好漂亮喔！」我問妹還要不要吃剛剛沒吃完的早餐粉，
妹答:「要」，接著又說：「可是我的衣服會用髒ㄋㄟ」
我說：「那穿圍兜兜好不好?」
妹說:「不用啦！」
2.氣球破掉了，妹說:「媽咪修」
我說:「破太大洞都裂開了沒辦法修了」
妹說:「用膠帶修啊！」
我說:「真的沒辦法修了，你把地上的氣球撿去丟垃圾筒，媽咪改天買比較不會破的氣球給你」
天熱，妹光著腳，拿著氣球碎片準備要走進廚房丟進垃圾桶，
我提醒妹:「沒穿拖鞋進廚房腳腳會髒髒喔」
我原本以為妹會走到客廳穿他自己的小脫鞋，沒想到，妹聽完我的提醒後，
竟然一腳跨進廚房－我的拖鞋上，穿著我的拖鞋，丟了氣球碎片後，又光著腳丫走回客廳。
<span style = 'color:red'>善用詞彙</span>
1.看我幫妹妹洗頭，問:「泡泡ㄌㄟ?」
我說:「沖到水杓裡啦！」  妹說:「我要吃」
我說:「不行，吃了會中毒，死翹翹唷！」
妹說:「白雪公主才可以吃」(因為他每天都聽爸爸講白雪公主吃毒蘋果的故事)
2.妹說：「我想給阿公抱抱」，「可是阿公臉上好多鬍子喔！要阿公刮乾淨」
我故意說:「可是阿公沒有刮鬍刀ㄟ」，妹答:「用爸爸的」
我說：「那你去台東的時候記得要帶刮鬍刀給阿公喔！」   妹答:「好」
<span style = 'color:red'>綜合老大這幾天的表現，真的是可圈可點。</span>

PS.老大已經會唱好多歌了(魯冰花、情花開)，有時父母沒空陪她玩，他還會自己翻故事書，講故事唷！</span>
