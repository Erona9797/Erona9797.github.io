# 牽手散步去～(1歲2個月)

文章日期：2011-11-07 08:11

星期六艷陽高照，晴空萬里，是出遊的好天氣，可是，因二寶午睡到四點半才起來，大寶又不太聽話，語帶惋惜的跟大寶說：「真的很可惜！因為你不乖，所以媽咪不帶你去疏洪公園看羊咩咩了」。大寶的爸爸說要騎摩托車載她去，但，堅持一定要先跑完操場才要載妹去，結果，下午五點多，天都黑了才出發...@@
大寶說：「爸爸帶她去看羊咩咩，沒有餵羊咩咩吃草，也沒有牽到狗狗，也沒有騎腳踏車，只有吃完布丁就回家了！」...@@
大寶出發後，我不想錯過好天氣，所以，就牽著二寶的手，一路散步走到重陽國小，牽著二寶的小手，知道他很小，可是，牽著二寶散步的感覺，卻比牽著大寶還"順"，一路上，二寶都緊抓著我的手指，就這樣跟著我一直走，到了國小操場，還繼續跟我玩跑步追逐的遊戲，直到去爬司令台的樓梯，在爬下樓梯的最高階時，二寶突然坐下來，原來，他累了～～ 辛苦了！ 這麼小的baby，竟可以這樣配合的走將近300公尺～ 回程，我就抱著二寶，幫她按摩一下大腿、小腿，妹妹因為癢，而笑的嘰嘰嘎嘎，超好玩的。 第一次牽手帶二寶散步的經驗，感覺好甜美，好幸福，我想～ 二寶的心裡應該也這樣覺得吧！ 只是比較累一點..|||
![[IMG_2262.jpg]]二寶的模仿能力很強，在口語表達方面，大人說什麼話，二寶就會跟著發出類似的聲音，
例如：「哎唷喂呀」、「嘿咻」、「哈啾」、「肉圓仔(台語)」、「阿肥仔(台語)」。
看到電梯按鈕，會說：「滴滴」
抬頭看到電燈，會說：「燈燈」
看著自己的照片會說：「晏晏」
看到姐姐的照片會說：「接姊」
看到外婆的照片會說：「阿嬤」
看到媽咪的照片會說：「媽媽」
想喝水時會說：「嗲嗲」
想喝奶時會說：「ㄋㄟ ㄋㄟ」
看到牙刷的時候會說：「牙牙」
<span style = 'color:blue'> 行為：</span>
1.喜歡搬重物(椅子和白柚)，一邊搬還會一邊吃力的說：「ㄟ咻」
2.睡醒，在床上恩恩ㄟㄟ幾聲，如果看到是阿嬤，就會露出「你來做什麼的表情」，然後，撇頭就睡回去，假如是看到媽咪過去，那就會一直耍賴的哭著要我躺下給他喝奶繼續睡，如果我沒理他，他再哭一下，也會自己睡回去。睡飽後，若發現沒人在旁邊，會自己爬下床，走到浴室或客廳找人。
3.會拿著故事書，篤定看著你並發出「ㄝㄝㄝ」的聲音，要我們講故事給他聽。
4.很會跑了，衝得很快很像快跌倒的樣子，有一次姊姊跟在妹妹的後面追逐，妹妹突然跌倒，姊姊來不及閃，就跌趴在妹妹的頭上，妹妹的門牙就流血了，哭得好傷心...  唉唉唉，兩個跑步不太穩的小朋友，卻這麼喜歡玩跑步追逐的遊戲，我看小寶的門牙遲早會撞斷的。
5.跟二寶說：把「玩具放桶子裡、把牙刷拿來給媽咪」小寶都聽得懂而且會去做。
6.我常將地上的頭髮或是小雜物撿起來丟到廚房的垃圾桶，二寶會學我蹲下來→撿起來→丟到垃圾桶的動作，這點，和大寶小時候一樣，身教真的很重要。
我拿小馬桶要給他坐的時候，他會故意往反方向跑給我追，感覺很淘氣，通常他會跑給我追時，坐馬桶噓尿不到2秒鐘二寶就會站起來，不想尿，但不保證等一下不會尿。但如果他是剛睡醒在床上賴床的樣子，看起來也是不想讓我抱去坐小馬桶，但這時就要堅持抱他坐，因為一下子就會尿很多在尿桶裡囉！
<span style = 'color:blue'>吃東西：</span>
大寶吃東西喜歡小小口，二寶吃東西喜歡大大口，給他一口大口的，他就邊吃邊玩，吃完再來吃一大口，常常三兩下二寶吃飽了，大寶才吃幾口而已。
<span style = 'color:blue'>刷牙：</span>
上星期，我要幫二寶刷牙，二寶堅持要自己拿牙刷，而且不讓我刷，如果把牙刷拿走，二寶就會賴在地上哭，我索性讓她哭個夠，一哭就是半小時，連續兩天過後，他再耍賴時，我就：「你要躺地上嗎?那就讓你躺吧！」二寶雖然還是耍賴的哭，但是不會一直躺在地上，馬上就爬起來要我抱了。
這星期，二寶會乖乖的開口讓我刷牙，也不會吵著要拿牙刷了，晚上若生氣想耍類，也不會真的躺在地上哭～～  看來，孩子是需要訓練的，而且大人一定要堅持，不可以，孩子吵就給糖吃，不可以一哭就順從小孩。
註:二寶的犬齒和臼齒都冒出頭來囉！
PS. 前陣子每天早上起床，就看到二寶的大腿和上臂出現一大堆被叮咬的包，大寶八個月的時候也曾這樣，上網查過後，應該是蕁麻疹，問題是，是什麼讓二寶過敏引發蕁麻疹呢？經我觀察後，也刻意放一隻螞蟻在妹的小腿上爬，雖然沒有馬上出現紅疹，但一小時後，爬過的地方就像其他地方一樣出現叮咬的膨疹了。
故判定是螞蟻過敏，將床單、棉被、枕頭套 都洗過曬太陽後，二寶就沒有被叮了。
