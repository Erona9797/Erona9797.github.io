# 口語表達能力進步許多~一歲五個月

文章日期：2012-02-16 22:18
![[IMG_7536.jpg]]
<span style = 'color:blue'>表達能力：</span>
回台東之前，二寶只會說兩個字的詞彙，例如：尿尿、菜菜、抱抱。回台東幾天後，妹開始會用三個字的詞彙，漸漸的，也能跟著模仿連續說出五個字的一句話。但能靈活運用的詞彙大多是三個字以內的，例如：魚仔、ㄅㄨㄅㄨ、給我、這邊，較特別的是，妹比較淘氣，會故意捉弄我，例如：天氣冷時，若我坐著露出後尾椎附近的皮膚時，二寶會故意用冰冰的小手，伸進我的後背，然後說：「冰冰」。偶爾會突襲我，突然將手伸進我胸口，然後說「ㄋㄟㄋㄟ」或說「摸ㄋㄟㄋㄟ」。有時又或故意做一些站高椅子爬高等高難度的動作，當有點失去平衡時，會說「阿咪娜」(救命哪)！
<span style = 'color:red'>服從命令：</span>
相較於大寶，妹妹也會耍脾氣，也很兇，若不順她，一下子就會哭到噴淚，但，我只要馬上問二寶說，「要不要乖乖聽話、想不想媽咪抱抱，要就不要哭，不哭才可以抱抱」。妹馬上就會不哭，然後走過來給我抱抱。
有時候妹只是不依，一直亂動我桌上的東西，我就說：「想不想媽咪抱抱？想就不要亂動媽咪的東西，乖乖坐好」，妹就會點頭，而且乖乖不會亂動。
二寶想抱抱或是想媽咪陪睡覺，我說：把地上的積木收好，或是，把床上的玩具收好，妹都會乖乖的收好，並歸位，真的超聽話的。
當姊姊想要跟媽咪睡時，那個晚上也會非常乖，會主動跑到床上收玩具，還會照顧妹妹，會自動自發做平常我要求她們該做的事情。
後記：
一歲六個月時，妹妹會跟著唱鳳飛飛的掌聲響起(有錄音、錄影為證)
看到姊姊跑步滑倒，妹妹會說：「姊姊，跑步跑步，跌倒」
兩個好動的寶貝，已不容易拍照，現在多以錄影的方式來為寶貝做成長紀錄。
大寶三歲三個月，小寶一歲六個月，天真可愛又活潑，現在照顧兩個小寶貝，感覺好幸福，好享受。
