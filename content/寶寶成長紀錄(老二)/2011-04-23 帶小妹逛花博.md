# 帶小妹逛花博

文章日期：2011-04-23 17:40
![[IMG_1480.jpg]]
4/22(五) 帶學生去花博
<span style = 'color:blue'>1.集合</span>
8:30 在圓山站前集合，廣場上早已排了長長的人龍，還好學生團體入口的人不多。等待開放的空檔，我讓學生們抱抱妹妹，妹被這麼多的大姊姊輪流抱來抱去，一直努力認人，也很想找媽咪，可是我有經驗了，趕快閃遠遠，不然她看到我一定會哭著要我抱，這樣學生就抱不到了～ 過了約10分鐘，學生叫著說，寶寶好像快哭了，我靠近一瞧，被妹看到了，妹馬上委屈的大哭，意思就是說：「我找你這麼久，你去哪裡了阿～怎嚜都找不到媽咪，人家想要媽咪抱抱喔喔了啦！」 我一邊安撫妹，一邊跟學生解釋說明媚妹哭的意思，妹聽了越哭越委曲，大夥兒聽了覺得好有趣，妹應該不解，為何她哭的這麼傷心，大家卻笑得這麼開心吧！
<span style = 'color:red'>2. 爭艷館</span>
拜團體進場由另一通道之賜，我們一下子就進入爭艷館，一下子美麗的花朵映入眼簾，由各式各樣台灣暢銷的外銷花卉排列出的各種主題，有主要外銷國的地標(新加坡、馬來西亞、紐約)，有生日party、婚紗、瓜棚、稻草人...，對花卉是外行人的我，只能看漂亮，拍拍照，然後前往下一個館排隊去。
<span style = 'color:red'>3.流行館～遠東環生方舟</span>
這個館就排了快了30分鐘，進去後看到東西～蠻失望的，我媽去年11月去看，所拍的照片到的內容，比我昨天看的還要豐富許多，不過，至少有看到「科技」、「環保」的產物～寶特瓶磚，還算勉強值得啦！
<span style = 'color:blue'>4.花茶殿</span>
由於跟學生約11點要在新生園區的花茶殿集合，但是看完流行館就已經快要11點了～心想不妙，想說搭接駁車，沒想到排隊人龍排到爭艷館前，只好背著妹妹，快步爬上人行路橋到美術園區，再走花之隧道前往新生園區的花茶殿，到現場已經11點半了，一群學生在等我～真不好意思。
<span style = 'color:green'>5.大佳園區</span>
中午時刻，台灣美食區人很多，看看我背著小孩能夠吃的食物實在有限，索性搭接駁車前往大佳園區，吃了一份「一口煎餃」50元，深海花枝燒50元，合作社的一口煎包才賣35元ㄟ～～
<span style = 'color:blue'>6.未來館、天使生活館</span>
回到新生園區，遇到學生，學生說這兩館還蠻值得看的，而且不會排很久，所以我就去排了，排了30分鐘後進場，未來館內是熱帶到溫帶的各種植物，還有茶業、永生花的製法介紹，還不賴。天使生活館就很爛了，就是用投影機投射在白窗簾布上，以及看七分鐘的3D劇場短片，就這樣而已，我以為會看到更科技的東西呢！
![[IMG_1346.jpg]]<span style = 'color:red'>7.美術園區(寰宇庭園)</span>
看完後，已下午四點了，想說在去美術園區，補拍一些上次來沒拍到的畫面，拍完再去圓山站搭車回家，回到家已經17點40，今天背妹逛花博總計10小時，逛了四個館，四個園區，腿好酸～幸好媚妹很配合，每兩小時喝一次奶，還在哺乳室內喝奶後才大便，可以馬上洗屁屁，到美術園區也很配合的拍照，整體來說，此次的花博行還蠻開心的啦！
