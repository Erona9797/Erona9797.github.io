# 老二似有超齡的反應-滿九個月的比較

文章日期：2011-06-18 21:48

二女兒與大女兒相較，似乎有更超齡成熟的表現。
<span style = 'color:blue'>照鏡子:</span>
兩姐妹都很喜歡照鏡子，姊姊是陶醉的看著自己，妹妹則是興奮好奇的比較鏡子裡面看到的畫面和實際有無異同。
<span style = 'color:purple'>鬧脾氣時:</span>
姊姊無論什麼情況，只要給他乳房，他就不哭鬧。妹妹則是要喝水就一定會不耐煩的吵到大人給他水喝；想睡覺時，早上就會找阿嬤帶他出去坐在推車上睡覺，晚上就一定會找媽咪吵著要睡覺。
<span style = 'color:red'>裝可愛：</span>
姊姊好像也是九個月左右會裝可愛，十個月時，是聽得懂爸媽說裝可愛他才裝可愛的斜著頭。妹妹目前聽不懂「裝可愛的指令」，但會自己把頭斜一邊裝可愛，而且還同時眨眼咪眼睛，感覺更淘氣。
<span style = 'color:green'>爬行和站立：</span>
姊姊剛滿九個月那天會爬，妹妹是滿九個月已經蠻會爬了，只是，還是不想爬，除非完全找不到東西讓他扶著站起來他才甘願用爬的，妹可以自己從坐在地上，不扶任何東西就站立超過40秒，這件事情，姊姊11個月時才辦得到。
<span style = 'color:red'>舌頭咑咑咑：</span>
大概滿五個月~六個月左右，就會用舌頭發出「咑」的聲音，大概七個月會坐的時候會較「爸爸」，現在(九個月)還是叫媽咪「ㄟ」。但已會透過「叫」發出聲音來表示「生氣、要媽咪看他這邊...」等意見了！
老大何時舌頭會咑咑咑忘了，一歲一個月會走路時才會叫爸爸，11個月時就會叫媽媽了。
阿嬤看到老二的表現，常說:「這麼小的孩子應該不會這樣...、上天是公平的。妹剛出生受這麼多的苦....」
註:
1.今天幫兩姐妹量體重，姊二歲半13公斤，妹九個月8公斤。
2.姊姊去圖書館看到中年媽媽跳舞，回家就一直請阿嬤和爸爸陪他跳舞，根據他的舞姿，我猜應該是探戈，待日後驗證。
3.姊會希望爸媽用照顧妹妹的方式照顧她，例如:抱著喝水、用滴管或湯匙喝水。
